# 🚦 AI Traffic Management System

## 📖 Overview
AI Traffic Management System is a web application developed using React.js and Firebase. The project helps manage traffic signals through a simple and user-friendly interface.

## ✨ Features
- User Login & Signup
- Firebase Authentication
- Dashboard
- User Profile
- Traffic Signal Management
- Add Traffic Signals
- Search Traffic Signals
- Delete Traffic Signals
- Responsive UI

## 🛠 Technologies Used
- React.js
- JavaScript
- HTML
- CSS
- Firebase
- React Router DOM
- GitHub

## 🚀 Installation
1. Clone the repository
2. Run:
```
npm install
```
3. Start the application:
```
npm start
```
4. Open:
```
http://localhost:3000
```
